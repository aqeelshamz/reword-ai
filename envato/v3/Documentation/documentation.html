🚀 Getting Started

To get started with RewordAI, you need to install the application on your server and configure the backend and frontend settings. The following sections will guide you through the installation process and provide detailed instructions for setting up the backend and frontend components.

🔧 Tech Used

RewordAI is built using modern web technologies and file formats, ensuring compatibility and flexibility for various use cases and environments. The application leverages the following technologies and file formats:

  • Node.js: JavaScript runtime for building scalable and efficient server-side applications.
  • Express: Fast, unopinionated, minimalist web framework for Node.js, providing a robust set of features for web and mobile applications.
  • MongoDB: NoSQL database used to store application data, providing flexibility and scalability for managing large volumes of data.
  • Next.js: React framework for building server-rendered applications, providing features such as server-side rendering and static site generation.
  • Tailwind CSS: Utility-first CSS framework for rapidly building custom designs, providing a flexible and modular approach to styling web applications.
  • DaisyUI: Tailwind CSS component library for building modern user interfaces, providing a collection of customizable components and utilities.
  • OpenAI GPT-4: Language model used for AI-powered evaluation of answer sheets, providing advanced natural language processing capabilities.

🔌 Installation

1. Node.js and NPM

Install the NPM package manager to manage the application's dependencies and run scripts. You need to have Node.js installed to use NPM. You can download and install Node.js from the official website https://nodejs.org/en/download


Download and install the latest version of Node.js, which includes NPM, the Node.js package manager. You can download the installer from the official website and follow the installation instructions provided for your operating system.


After installing Node.js, you can verify the installation by running the following commands in your terminal or command prompt:


node --version
v20.11.0

2. MongoDB (Database)

Install MongoDB, a NoSQL database used to store application data. You can download and install MongoDB from the official website https://www.mongodb.com/try/download/community


Download and install the latest version of MongoDB, following the installation instructions provided for your operating system. After installing MongoDB, you can start the MongoDB server by running the following command in your terminal or command prompt:


mongod

You can also use MongoDB Atlas, a cloud-based database service, to host your MongoDB database. MongoDB Atlas provides a free tier for development and testing purposes, allowing you to create a cloud-hosted MongoDB database without the need to install and manage a local database server.

https://www.mongodb.com/atlas/database

⚙️ Backend Setup

After installing Node.js and MongoDB, you can proceed with setting up the backend of RewordAI. The backend is responsible for handling API requests, managing data, and performing AI-powered evaluation of answer sheets. Follow the steps below to configure the backend:


Navigate to the backend directory (Projects/backend) of the RewordAI project. This directory contains the backend codebase, including the Node.js application and API server.

1. Install Dependencies

Install the required dependencies for the backend by running the following command in your terminal or command prompt:


npm install

2. Open the directory in your code editor

Open the backend directory in your preferred code editor. You can use Visual Studio Code, Sublime Text, Atom, or any other code editor of your choice. If you are using Visual Studio Code, you can open the directory by running the following command from the backend directory in your terminal or command prompt:


code .

3. Configure Environment Variables

Open the .env file in the backend directory and configure the environment variables based on your environment and preferences. The .env file contains settings such as the database connection URI, server port, and API keys.



Replace the following placeholders with your actual values:

  • DB_URL: MongoDB connection URI, including the database name.
  • OPENAI_API_KEY: OpenAI API key for accessing the GPT-4 language model.

    • RewordAI uses the OpenAI GPT-4 language model for AI-powered evaluation of answer sheets. You can obtain an API key from https://platform.openai.com/api-keys and use it to access the GPT-4 language model for evaluating answer sheets.



  • TOKEN_SECRET: Secret key for generating JSON Web Tokens (JWT).
  • STRIPE_SECRET_KEY: Secret key for integrating Stripe payments.

    • You can sign up for a free account on Stripe and obtain the publishable key for your application. https://dashboard.stripe.com/

  • RAZORPAY_KEY_ID: Key ID and secret for integrating Razorpay payments.

    • You can sign up for a free account on Razorpay and create a new key to obtain the key ID for your application. https://dashboard.razorpay.com/app/website-app-settings/api-keys

  • RAZORPAY_KEY_SECRET: Key secret for integrating Razorpay payments.
  • PAYPAL_CLIENT_ID: Client ID for integrating PayPal payments.

    • You can sign up for a free account on PayPal Developer and create a sandbox application to obtain the client ID and secret for your application. https://developer.paypal.com/dashboard/applications/sandbox

  • PAYPAL_CLIENT_SECRET: Client secret for integrating PayPal payments.
  • APP_NAME: Name of the application (e.g., RewordAI).
  • SMTP_HOST: SMTP host for sending email notifications (e.g., smtp.gmail.com).
  • SMTP_PORT: SMTP port for sending email notifications (e.g., 465).
  • SMTP_USER: SMTP username for sending email.
  • SMTP_PASS: SMTP password for sending email.

4. Configure Utils

Open the backend/utils directory in the backend directory and configure the following variables in the util.js file:

Replace the following placeholders with your actual values:

  • currency: Currency code for payment transactions (e.g., "inr").
  • paypalCurrency: Currency code for PayPal transactions (e.g., "USD").
  • paymentMethods: Array of payment methods supported by the application (e.g., ["stripe", "razorpay", "paypal"]).
  • merchantName: Name of the merchant or application.
  • merchantAddress: Address of the merchant or application.
  • razorpayThemeColor: Theme color for the Razorpay payment gateway (e.g., "#528ff0").

5. Start the Backend Server

Start the backend server by running the following command in your terminal or command prompt:


npm start

🖥️ Frontend Setup

After configuring the backend, you can proceed with setting up the frontend of RewordAI. The frontend is responsible for providing a user interface to interact with the application, including features for class and exam management, answer sheet evaluation, and result analysis. Follow the steps below to configure the frontend:


Navigate to the frontend directory (Projects/frontend) of the RewordAI project. This directory contains the frontend codebase, including the React application and user interface components.

1. Install Dependencies

Install the required dependencies for the frontend by running the following command in your terminal or command prompt:


npm install

2. Open the directory in your code editor

Open the frontend directory in your preferred code editor. You can use Visual Studio Code, Sublime Text, Atom, or any other code editor of your choice. If you are using Visual Studio Code, you can open the directory by running the following command from the frontend directory in your terminal or command prompt:


code .

3. Configure Environment Variables

Open the .env.local file in the frontend directory and configure the environment variables based on your preferences.



Replace the following placeholders with your actual values:

  • NEXT_PUBLIC_PAYPAL_CLIENT_ID: Client ID for integrating PayPal payments. (refer backend setup)

4. Configure Utils


Open the frontend/src/utils directory in the frontend directory and configure the following variables in the utils.ts file:



Replace the following placeholders with your actual values:

  • serverURL: Backend server URL for API requests (e.g., "http://localhost:8080").
  • currencySymbol: Currency symbol for payment transactions (e.g., "₹").
  • appName: Name of the application (e.g., "RewordAI").
  • stripeKey: Your Stripe Key (refer previous section for stripe setup).

5. Start the Frontend Server

Start the frontend server by running the following command in your terminal or command prompt:


npm run dev
